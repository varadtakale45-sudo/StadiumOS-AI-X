# StadiumOS AI X

StadiumOS AI X is an AI-powered autonomous stadium command center designed to help event organizers monitor, manage, and respond to stadium operations in real time.

The platform provides a modern dashboard that visualizes crowd activity, security alerts, emergency responses, infrastructure health, weather conditions, and operational analytics through an intuitive interface.

This project was built as a hackathon prototype to demonstrate how AI can enhance stadium safety, operational efficiency, and decision-making.

---

## Problem

Managing a large stadium during sporting events, concerts, or public gatherings involves thousands of moving parts.

Operators must simultaneously monitor:

- Crowd congestion
- Security incidents
- Emergency responses
- Infrastructure health
- Weather conditions
- Resource allocation

Traditional systems often rely on disconnected dashboards and manual coordination, making it difficult to react quickly during critical situations.

---

## Solution

StadiumOS AI X combines operational intelligence into a single command center.

The platform enables operators to:

- Monitor live stadium operations
- Track crowd movement
- View AI-generated alerts
- Analyze operational metrics
- Monitor emergency incidents
- Visualize stadium-wide activity
- Access infrastructure status
- Improve decision making through data-driven insights

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/stadiumos-ai-x.git
```

Move into the project

```bash
cd stadiumos-ai-x
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
